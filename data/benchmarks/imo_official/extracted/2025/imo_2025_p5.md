# IMO 2025 Problem 5

- Source: International Mathematical Olympiad official English PDF
- Source PDF: `data/benchmarks/imo_official/problems_pdf_en/IMO-2025-problems-eng.pdf`
- Day: 2
- Points: 7

Alice and Bazza are playing the inekoalaty game, a two-player game whose rules depend on a positive real number lambda which is known to both players. On the nth turn of the game, starting with n = 1, the following happens:

- If n is odd, Alice chooses a nonnegative real number x_n such that

```text
x_1 + x_2 + ... + x_n <= lambda n.
```

- If n is even, Bazza chooses a nonnegative real number x_n such that

```text
x_1^2 + x_2^2 + ... + x_n^2 <= n.
```

If a player cannot choose a suitable number x_n, the game ends and the other player wins. If the game goes on forever, neither player wins. All chosen numbers are known to both players.

Determine all values of lambda for which Alice has a winning strategy and all those for which Bazza has a winning strategy.
