# IMO 2025 Problem 3

- Source: International Mathematical Olympiad official English PDF
- Source PDF: `data/benchmarks/imo_official/problems_pdf_en/IMO-2025-problems-eng.pdf`
- Day: 1
- Points: 7

Let N denote the set of positive integers. A function f : N -> N is said to be bonza if

```text
f(a) divides b^a - f(b)^f(a)
```

for all positive integers a and b.

Determine the smallest real constant c such that f(n) <= c n for all bonza functions f and all positive integers n.
