# IMO Official Problems Archive

Source: https://www.imo-official.org/problems.aspx

This directory stores the official English PDF problem statements for all IMO
years listed on the official problems archive at download time.

- Years covered: 1959-2025
- Count: 66 PDFs
- Note: 1980 is not present because there was no IMO in 1980.

Files:

- `problems_pdf_en/`: official English problem PDFs, one per year.
- `year_pages_html/`: downloaded official year pages used to discover PDF URLs.
- `problems_index.html`: downloaded official problems index page.
- `years.txt`: years discovered from the official index.
- `manifest.tsv`: source URL, local path, status, file size, and SHA-256 for each PDF.
